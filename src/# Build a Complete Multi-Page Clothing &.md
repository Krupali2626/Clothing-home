# Build a Complete Multi-Page Clothing & Home Appliances eCommerce Website

Create a **modern, premium, responsive, SEO-friendly multi-page eCommerce website** using only the following technologies:

* React (JavaScript only)
* React Router DOM
* React Bootstrap
* React Icons
* Swiper.js (for all sliders/carousels)
* Axios (for API ready structure)
* Bootstrap Icons (optional)
* CSS Modules are NOT allowed.
* Styled Components are NOT allowed.
* Tailwind CSS is NOT allowed.
* Material UI is NOT allowed.

## Naming Rules

* Every custom CSS class must start with:

```
d_
```

Example

```
d_header
d_banner
d_product_card
d_footer
d_btn_primary
```

No class should exist without the prefix.
